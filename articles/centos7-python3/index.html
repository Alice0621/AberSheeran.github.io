编译安装Python3

值得注意的是，使用以下步骤能在最新版的CentOS桌面版安装成功Python3。但是当我使用以下步骤在云服务器上时，失败了。

安装编译环境

 
xxxxxxxxxx
yum install yum-utils
#安装编译python的编译环境以及依赖包
yum-builddep python

下载Python3安装包

 
xxxxxxxxxx
#如果下载太慢，可以换国内镜像，自行百度
wget https://www.python.org/ftp/python/3.5.2/Python-3.5.2.tgz

解压并编译安装

 
xxxxxxxxxx
tar -xzvf Python-3.5.2.tgz -C /usr/local/Python3
cd /usr/local/Python3/Python-3.5.2
./configure
make
make install

使用Python3和pip3

只需要使用python3和pip3即可。

使用yum安装Python3

经过测试，在腾讯云的CentOS7服务器上使用这种方法可以直接安装Python3

 
xxxxxxxxxx
yum -y update
yum install -y https://centos7.iuscommunity.org/ius-release.rpm

然后直接使用yum即可，例子如下

 
xxxxxxxxxx
yum install -y python35u python35u-libs python35u-devel python35u-pip

使用也很简单，打出命令python3.5或者pip3.5即可。 如果需要其他版本，譬如3.4，只需要把以上命令中的所有的35改成34即可。
